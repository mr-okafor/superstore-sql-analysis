# Superstore Sales SQL Analysis

## Project Overview
This project contains SQL queries used to analyze the Tableau Superstore dataset.

The aim of the analysis is to understand sales performance, profitability, and customer behavior using SQL.

## Objectives
- Analyze total sales performance
- Identify the most profitable product categories
- Understand regional sales distribution
- Evaluate customer segments

## Tools Used
- MySQL
- SQL
- GitHub

## Project Status
SQL analysis completed. Data visualization in Power BI is ongoing.
